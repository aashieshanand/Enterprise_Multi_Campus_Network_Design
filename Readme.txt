# Enterprise Multi-Campus Network Design & Configuration

## Project Overview
This project demonstrates the design and configuration of a secure, scalable enterprise multi-campus network following CCNA best practices.

## Network Architecture
- Core: Routers (Main, Branch, Cloud)
- Distribution: Layer 3 switches
- Access: Departmental access switches

## Technologies Used
- Cisco IOS
- VLAN & Inter-VLAN Routing
- RIP v2
- DHCP
- SSH Security

## Features Implemented
- VLAN segmentation
- Router-on-a-Stick
- WAN routing using RIP v2
- Centralized DHCP
- Secure remote access

## Testing & Validation
- Inter-VLAN ping tests
- WAN reachability
- DHCP verification
- SSH access verification
